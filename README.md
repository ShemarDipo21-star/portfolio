# Shemar Dipotaroeno — Portfolio

Personal portfolio website for Shemar Dipotaroeno, ICT Infrastructure Technician based in Para, Suriname.

## Pages

- **Home** — Hero introduction with photo
- **About** — Background, personal details and contact info
- **Education** — Academic history (NATIN, NATIN-MBO, University)
- **Experience** — Work history (Itee NV, Staatsolie, ECIS, Smartwigi)
- **Skills** — Technical skills with progress bars
- **Certifications** — CompTIA certification path
- **Contact** — Email, LinkedIn and phone

## Tech

Built with plain HTML, CSS and JavaScript. No frameworks or dependencies. Hosted via GitHub Pages.

## Structure

```
index.html / about.html / education.html / experience.html / skills.html / certifications.html / contact.html
css/        — base.css + one CSS file per page
js/         — script.js
img/        — shemar.jpg
```
